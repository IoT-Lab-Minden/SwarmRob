Allocate services to workers using the push-relabel algorithm specified URL

Parameters

  • services – Services that should be allocated

  • workers – Workers that should carry out the services

  • hardware_matrix – Binary matrix where each entry specifies if a service i can be executed on a worker j

  • cost_matrix – Matrix where each entry describes the cost for running a service i on a specific worker j

  • capacity_matrix – Matrix where each entry describes the capacity of a worker
